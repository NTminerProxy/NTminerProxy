### 👉 **Linux安装**

   <p>&emsp;&emsp;运行以下shell指令以运行工具包</p>

   ```sh
bash <(curl -s -L https://github.com/NTminerProxy/NTminerProxy/raw/master/ntinstall.sh)
   ```
   
   <p>&emsp;&emsp;成功运行后，您将看到以下菜单, 根据提示安装即可。</p>
